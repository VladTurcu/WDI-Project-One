How to play


The SnakeStein is a Math[:)] based game with a different operation to perform on each level.

Level 1

Addition
The next number you have to get is the sum of all previous numbers
Example: c = a + b

Level 2

Multiplying
You will discover the next number by multiplying the last two
Example: c = a * b
